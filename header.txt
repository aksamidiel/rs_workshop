В данном модуле студентам необходимо:
Получить общее представление о системе контроля версий Git.
Начать работу с Git на примере веб-сервиса GitHub.
Познакомиться с терминологией:
Repository: Local, Remote/Origin
Working Directory
Staging Area
Познакомиться c основными командами (clone, add, commit, push, pull)
Познакомиться c понятием "ветка" в git: создание новой, слияние, перебазирование
Познакомиться с Markdown, оформить страницу
Создать собственную веб-страничку при помощи github-pages
